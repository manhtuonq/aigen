AI Tạo Ảnh & Video
Mô tả ý tưởng → Claude AI tối ưu prompt → Tạo ảnh tự động → Xuất video
Claude API Key (bắt buộc để AI tối ưu prompt)
Lấy API key miễn phí tại console.anthropic.com. Key được lưu trong bộ nhớ tạm của trình duyệt, không gửi đi đâu khác.
⚠️ Nếu không có key: hệ thống sẽ dùng prompt tiếng Anh tự động (không cần Claude).
Lưu ý CORS khi chạy local: Anthropic API không cho phép gọi trực tiếp từ trình duyệt (CORS). Nếu bạn chạy file local (127.0.0.1), Claude API sẽ bị chặn — hệ thống sẽ tự dùng chế độ Fallback (không cần Claude) và vẫn tạo ảnh bình thường qua Pollinations.ai. Để dùng Claude đầy đủ, hãy deploy lên server thật hoặc dùng proxy.

AI Tạo Ảnh & Video
  1. Vào mục AI Ảnh & Video
  2. Dán Claude API key vào ô key (lấy tại console.anthropic.com)
  3. Gõ mô tả (tiếng Việt hoặc tiếng Anh)
  4. Chọn style, số khung hình và tỉ lệ
  5. Nhấn Tạo khung hình AI
  6. Tải ZIP hoặc render thành Video GIF/WebM
Lưu ý CORS: Khi chạy file HTML local (file:// hoặc 127.0.0.1), Claude API bị chặn bởi trình duyệt. Hệ thống sẽ tự động dùng Fallback mode — vẫn tạo ảnh được qua Pollinations.ai. Để dùng Claude API đầy đủ, cần deploy lên hosting thật (Netlify, Vercel, GitHub Pages...).
